# Too Good to be True ✨

An interactive, scroll-driven website blending smooth scrolling and engaging animations.

## Overview 🌟
Built with:
- **HTML5 & CSS3** 🖥️
- **JavaScript** 🚀
- **GSAP (ScrollTrigger)** 🎬
- **Locomotive Scroll** 🏃‍♂️

## Features 🚀
- Smooth, custom scrolling
- Scroll-triggered animations
- Fun custom cursor
- Responsive design
